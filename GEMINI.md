# EduSmart Landing Page Planning

## Directory Overview

This directory serves as the foundational planning and documentation workspace for the "EduSmart" project. EduSmart is an online education platform, and this specific project focuses on designing and building a conversion-optimized landing page MVP for digital marketing campaigns (Google Ads & Facebook Ads). The directory currently holds non-code assets and primarily product requirements, intended to guide the future technical implementation.

## Key Files

- **`PRD.md`**: The Product Requirements Document. This is the central source of truth for the project. It outlines the project goals, target audience, structural framework (AIDA), technical stack specifications (Svelte/SvelteKit, Tailwind CSS), dummy data for courses and mentors, digital marketing strategies, and Key Performance Indicators (KPIs).

## Usage

The contents of this directory are intended to be used as instructional context and a roadmap for AI-assisted development.

- Use `PRD.md` to understand the business goals, UI/UX requirements, and technical stack (SvelteKit) before generating any code.
- As the project evolves into a codebase (e.g., when `package.json` and `src/` are added), this `GEMINI.md` file should be updated to reflect the new code structure, build commands, and development conventions.
